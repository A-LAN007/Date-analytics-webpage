# 智能数据看板 - DataVision

> 一个面向业务决策者的多页面数据分析仪表盘，支持模拟实时数据、阈值预警、归因分析、任务追踪及趋势预测。

## ✨ 功能特性

- **6 大页面**：总览看板、归因分析、预警中心、行动任务、预测复盘、明细报表
- **动态数据模拟**：支持按日/周/月/同比切换，一键随机刷新数据
- **实时模式**：开启后数据自动微扰动，模拟真实业务波动
- **阈值告警**：可自定义华南区销售额、CAC、转化率阈值，并自动标红
- **趋势预测**：基于线性回归预测未来3日 GMV，并绘制在图表上
- **风险指数**：综合多指标生成风险评分及进度条
- **完全前端实现**：纯HTML/CSS/JS + Chart.js，无后端依赖

## 🛠️ 技术栈

- JavaScript (ES6)
- Chart.js 4.4.0

## 📌 使用说明

- **时间筛选**：点击“今日/本周/本月/累计同比”按钮，数据自动按时间尺度缩放。
- **随机刷新**：点击右上角“随机刷新”按钮，重新生成一组业务数据。
- **实时模式**：打开导航栏的“实时模式”开关，数据会每隔2.8秒微调一次。
- **阈值配置**：点击“阈值”按钮，可自定义华南区环比、CAC、转化率的告警阈值。
- **查看详情**：点击任意卡片、图表或漏斗步骤，会弹出模态框显示详细分析。
- **导出数据**：点击“导出”按钮，可将当前主要指标保存为CSV文件。

## 📈 数据逻辑说明

- 所有数据均为前端模拟生成，展示效果真实。
- GMV、订单量、用户数等指标会根据所选时间范围自动缩放（日/周/月/同比）。
- 趋势图包含3日移动平均（MA3）和5日移动平均（MA5），以及基于线性回归的未来3天预测。
- 预警规则：华南区销售额环比下降超过阈值、CAC超过上限、转化率低于下限时自动标红。
